## QR Code Generator App

The QR Code Generator App is a simple and interactive web application built using **Python and Streamlit** that allows users to generate QR codes instantly from any text or URL.

This project demonstrates how Python libraries can be integrated with a modern web UI to build real-world utility applications.

---

## Key Features

- Generate QR codes from any text or URL
- Customizable QR code colors (foreground & background)
- Real-time QR code generation with instant preview
- Download generated QR codes as PNG images
- Clean and user-friendly Streamlit interface
- Fully browser-based application (no installation required for end users)

---

## Tech Stack

- Python
- Streamlit
- qrcode library
- Pillow (PIL)

---

## Project Objective

The objective of this project is to build a lightweight and efficient QR code generator that provides a simple interface for users to quickly convert any information into a scannable QR code.

It also demonstrates skills in:
- Python GUI development using Streamlit
- Working with image processing libraries
- Real-time user interaction handling

---

## Use Cases

- Sharing links instantly

- Event or product information sharing
